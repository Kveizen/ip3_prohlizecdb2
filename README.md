# ip3_prohlizecdb2

1. Pro plné fungování je potřeba doplnit kód v **config_local.json** a **config.json**</br>
{
  "db": {
    "user" : "váš uživatel",
    "password" : "vaše heslo"
  }
}
</br>

2. Dále je potřeba nainstalovat composer a v terminalu spusti: **composer install** a **composer update**
